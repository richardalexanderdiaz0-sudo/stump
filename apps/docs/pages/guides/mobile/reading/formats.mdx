# Supported formats

The mobile app doesn't yet support all the formats that your Stump server supports. The following table provides an overview of formats and their status:

| Format  | Streaming | Offline | Notes                                                 |
| ------- | --------- | ------- | ----------------------------------------------------- |
| CBZ/ZIP | ✅        | ✅      |                                                       |
| CBR/RAR | ✅        | ❌      |                                                       |
| EPUB    | ❌        | ✅      |                                                       |
| PDF     | ⚠️        | ✅      | Some PDF pages are blurry when using streaming reader |

## OPDS Formats

You may download books from your OPDS catalogs and read them offline, however only some formats support a streaming reader. The following table provides an overview of formats and their status:

| Format  | Streaming | Offline | Notes                                                 |
| ------- | --------- | ------- | ----------------------------------------------------- |
| CBZ/ZIP | ✅        | ✅      |                                                       |
| CBR/RAR | ✅        | ❌      |                                                       |
| EPUB    | ❌        | ✅      |                                                       |
| PDF     | ⚠️        | ✅      | Some PDF pages are blurry when using streaming reader |
